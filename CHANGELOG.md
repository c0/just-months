## [1.0.0] - 2026-03-22

### Added
- Initial release
- macOS menu bar calendar widget showing three months (previous, current, next)
- Today highlighted with coral accent block cursor
- Retro terminal aesthetic: monospace, dark navy, cyan phosphor text
- WidgetKit extension for Notification Center widget
- Light and dark mode support
